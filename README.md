### Quick start
```bash
# clone our repo
$ git clone https://github.com/brood915/ngBlog.git

# change directory to your app
$ cd ngBlog

# install the dependencies with npm
$ npm install

# start the server
$ npm start

# start the app
$ Go to http://localhost:8080 in your browser.
```
